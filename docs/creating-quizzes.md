# クイズ作成・追加マニュアル

TEKITOブログに新しいクイズを追加する手順を説明します。
データ形式は **CSV** です。ExcelやGoogleスプレッドシートなどで編集可能です。

## 手順概要

1. `data/quizzes/` フォルダを開く
2. 新しいCSVファイルを作成する（例: `math-new.csv`）
3. 規定のフォーマットでクイズデータを記述する
4. 保存すると、自動的にクイズ一覧ページに表示されます

## データフォーマット

### 1. ファイル構成

CSVファイルは以下の3パートで構成されます：

1. **1行目: メタデータ行**（必須）
   - フォーマット: `#Meta,id,title,category,difficulty`
2. **2行目: ヘッダー行**（必須・スキップされます）
   - フォーマット: `id,question,option1,option2,option3,option4,correct,explanation`
3. **3行目以降: 問題データ行**
   - 1行につき1問を記述します。

### 2. テンプレート

以下のテキストをコピーして `.csv` ファイルとして保存してください。

```csv
#Meta,quiz-unique-id,クイズのタイトル,カテゴリ,Easy
id,question,option1,option2,option3,option4,correct,explanation
q1,問題文をここに記述します,選択肢1,選択肢2,選択肢3,選択肢4,1,解説文をここに記述します。
q2,2問目の問題文...,A,B,C,D,2,2問目の解説...
```

### 3. 各項目の説明

#### 1行目: メタデータ

| 列順 | 項目名 | 説明 | 例 |
| --- | --- | --- | --- |
| 1 | マーカー | 必ず `#Meta` と記述 | `#Meta` |
| 2 | クイズID | URLに使われます。ファイル名と同じ推奨 | `math-func` |
| 3 | タイトル | 画面に表示されるタイトル | `二次関数の基礎` |
| 4 | カテゴリ | 教科など | `数学` |
| 5 | 難易度 | `Easy`, `Medium`, `Hard` のいずれか | `Easy` |

#### 3行目以降: 問題データ

| 列順 | 項目名 | 説明 | 例 |
| --- | --- | --- | --- |
| 1 | 問題ID | クイズ内でユニークなID | `q1` |
| 2 | 問題文 | 問題のテキスト | `1+1は？` |
| 3 | 選択肢1 | 1つ目の選択肢 | `2` |
| 4 | 選択肢2 | 2つ目の選択肢 | `3` |
| 5 | 選択肢3 | 3つ目の選択肢 | `4` |
| 6 | 選択肢4 | 4つ目の選択肢 | `5` |
| 7 | 正解番号 | 正解の選択肢番号（**1〜4**） | `1` |
| 8 | 解説 | 解答後に表示される解説 | `正解は2です。` |

## 注意事項

- **文字コード**: UTF-8 で保存してください。
- **カンマ**: データ内にカンマ `,` を含める場合は、セル全体をダブルクォート `"` で囲んでください（Excel等は自動でやってくれます）。
- **正解番号**: プログラム内部では 0 始まりですが、CSV上では **1 始まり**（1, 2, 3, 4）で指定してください。

## サンプル

`data/quizzes/science-basics.csv` の例:

```csv
#Meta,science-basics,理科：原子と分子,理科,Easy
id,question,option1,option2,option3,option4,correct,explanation
q1,水の化学式は？,HO,H2O,HO2,H2O2,2,水分子は水素原子2つと酸素原子1つからなるため、H2Oです。
```
